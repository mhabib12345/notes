Setting up zram in Artix Linux

April 3, 2025 | Estimated reading time: 1 minutes

Here is the step for configuring zram in Artix Linux

  1. Create /etc/modules-load.d/zram.conf
zram
  1. Create /etc/modprobe.d/zram.conf
options zram num_devices=1

  1. Create /etc/udev/rules.d/10-zram.rules

    bash 
    ACTION=="add", KERNEL=="zram0", ATTR{initstate}=="0", ATTR{comp_algorithm}="zstd", ATTR{disksize}="4G", RUN="/usr/bin/mkswap -U clear %N"

  2. Finally, add zram to fstab for automount Edit /etc/fstab

/dev/zram0 none swap defaults,discard,pri=100 0 0

  1. Reboot to take the effect immediately

  2. After reboot, Check output with

zramctl
  1. Voila!!
